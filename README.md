DocCheck — Система контроля движения первичной документации

DocCheck — это специализированное приложение на Blazor, предназначенное для мониторинга и верификации бумажных документов (ТОРГ-12, УПД, акты) на всех этапах их пути: от экспедитора до передачи в архив или бухгалтерию.
📦 Проблема, которую мы решаем

В процессе доставки товара документы часто оформляются некорректно: отсутствуют печати, подписи не уполномоченных лиц или неверные даты. Это приводит к задержкам оплат и проблемам с налоговой отчетностью. DocCheck позволяет оцифровать путь бумажного носителя и вовремя заметить проблему.

🛠 Как это работает

Процесс построен на цепочке ответственности. Каждый этап фиксируется простым сканированием штрихкода документа:

    Водитель/Экспедитор: Сдает пакет документов после рейса.

    Склад / Приемка: Первичная проверка наличия.

    Менеджер: Детальная проверка корректности заполнения (печать, подпись, дата).

    Бухгалтер: Финальное подтверждение и прием в учет.

Если на любом из этапов обнаружена ошибка, документ помечается как «Проблемный» с указанием причины, что позволяет быстро вернуть его на исправление.
✨ Ключевые функции

    Трекинг в реальном времени: Видно, у кого именно сейчас находится конкретный документ.

    Быстрая регистрация: Использование сканера штрихкодов для перемещения документа между статусами/ролями.

    Контроль ошибок: Фиксация типовых нарушений (нет печати, не та подпись, исправления в тексте).

    Ролевая модель: Разграничение прав доступа (Склад, Менеджер, Бухгалтер).

    История изменений: Полный аудит того, кто и когда принимал или отклонял документ.

💻 Технологии

    Frontend: Blazor (Server/WASM) — для быстрого и интерактивного интерфейса.

    Backend: ASP.NET Core API.

    Database: Entity Framework Core.

    Интеграция: Поддержка работы со сканерами штрихкодов (как эмуляция клавиатуры).

🚀 Быстрый старт

    Клонирование:
    Bash

    git clone https://github.com/igmo71/DocCheck.git

    Настройка:
    Укажите параметры подключения к БД в appsettings.json.

    Запуск:
    Bash

    dotnet run --project DocCheck.Server
